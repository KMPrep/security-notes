Q6. Two-Pointer: Pair Sum in Threat Scores
Given a sorted list of threat scores and a target, find all pairs that sum to the target.

  • Return all pairs (not just one). Deduplicate.
  • Input: [1,2,3,4,5,6,7], target=8 → [(1,7),(2,6),(3,5)]
  • Complexity: O(n) two-pointer. No hash map.

Q7. Stack: Evaluate Nested DENY/ALLOW Policies
A policy evaluator processes lines like:

PUSH context=internal
ALLOW proto=TCP port=443
DENY proto=TCP port=80
POP
ALLOW proto=UDP port=53

Simulate a stack of contexts. Within each context, return a list of its rules.
On POP, rules from that context are finalized. Output rules grouped by context depth.

Q8. Regex: Multi-Pattern Log Scanner
Given 5 detection rules (compile them), scan a 100-line log file and report:

  • Which rules matched, on which lines.
  • Any line that matched 2 or more rules simultaneously (compound detection).
    Rules to use:
rules = {
    "ssh_brute":  r"Failed password for \w+ from [\d.]+",
    "sudo_abuse": r"sudo:.*TTY=[\w/]+ ; USER=root ; COMMAND=",
    "cron_mod":   r"(crontab|/etc/cron)",
    "file_exfil": r"scp|rsync|curl.*-o|wget.*-O",
    "privesc":    r"(chmod|chown)\s+(777|4755|u\+s)",
}

Q9. Dict: LRU Cache from Scratch
Implement an LRU (Least Recently Used) cache using only a Python dict and a list.

  • get(key): return value or -1 if not found; mark key as recently used.
  • put(key, value): insert; evict least recently used if capacity exceeded.
  • Capacity: 3 entries.
  • Test sequence: put(1,1), put(2,2), get(1), put(3,3), get(2), put(4,4), get(1), get(3), get(4)
  • Bonus: implement the O(1) version using collections.OrderedDict.

Q10. Binary Search: Find Minimum in Rotated Sorted Array
Given a rotated sorted array with no duplicates, find the minimum element.

  • Input: [3,4,5,1,2] → 1
  • Input: [11,13,15,17] → 11 (not rotated)
  • Complexity: O(log n).

Pattern 1: Dict Counting

from collections import defaultdict, Counter

# Manual
freq = {}
for item in data:
    freq[item] = freq.get(item, 0) + 1

# Pythonic
freq = Counter(data)

# Top 3
top3 = freq.most_common(3)

Pattern 2: Two-Pointer

# Pair sum in sorted array
lo, hi = 0, len(arr) - 1
while lo < hi:
    s = arr[lo] + arr[hi]
    if s == target:
        result.append((arr[lo], arr[hi]))
        lo += 1; hi -= 1
    elif s < target:
        lo += 1
    else:
        hi -= 1

Pattern 3: Sliding Window

# Fixed size window
window_sum = sum(arr[:k])
max_sum = window_sum
for i in range(k, len(arr)):
    window_sum += arr[i] - arr[i - k]
    max_sum = max(max_sum, window_sum)

# Variable window (longest without repeating)
left = 0
seen = {}
best = 0
for right, char in enumerate(s):
    if char in seen and seen[char] >= left:
        left = seen[char] + 1
    seen[char] = right
    best = max(best, right - left + 1)

Pattern 4: Stack

# Balanced brackets
stack = []
pairs = {')':'(', ']':'[', '}':'{'}
for c in s:
    if c in '([{':
        stack.append(c)
    elif c in ')]}':
        if not stack or stack[-1] != pairs[c]:
            return False
        stack.pop()
return not stack

# Monotonic stack (next greater element)
stack = []
result = [-1] * len(nums)
for i, n in enumerate(nums):
    while stack and nums[stack[-1]] < n:
        result[stack.pop()] = n
    stack.append(i)

Pattern 5: Binary Search

# Standard
lo, hi = 0, len(arr) - 1
while lo <= hi:
    mid = (lo + hi) // 2
    if arr[mid] == target: return mid
    elif arr[mid] < target: lo = mid + 1
    else: hi = mid - 1
return -1

# First occurrence
lo, hi, result = 0, len(arr) - 1, -1
while lo <= hi:
    mid = (lo + hi) // 2
    if arr[mid] == target:
        result = mid
        hi = mid - 1  # keep going left
    elif arr[mid] < target: lo = mid + 1
    else: hi = mid - 1
return result

Pattern 7: Big O Cheat Sheet

O(1)       — dict/set lookup, list index, stack push/pop
O(log n)   — binary search, heap insert in size-k heap
O(n)       — single loop, hash build, sliding window
O(n log n) — sort, merge sort, heap build from scratch
O(n²)      — nested loops, brute-force pair scan
O(2ⁿ)      — naive recursion without memoization (avoid)

Space:
O(1)   — two-pointer, in-place sort
O(n)   — hash map, result list, recursion call stack (depth n)
O(log n) — recursion call stack (depth log n, e.g., binary search)
