"""
=============================================================================
Q06 [Hard] — Detect Distributed Password Spray
=============================================================================

PROBLEM
-------
Password spraying differs from brute-force:
  - Brute-force: many passwords tried against ONE user from ONE IP.
  - Password spray: ONE (or few) common passwords tried against MANY users,
    from MANY different IPs (to stay under per-IP lockout thresholds).

Given a stream of authentication events, detect a coordinated spray campaign
even when individual source IPs each attempt only 1–3 users.

Key signals:
  1. Many distinct IPs each fail auth on a small number of users.
  2. The failed usernames share a common password (if available).
  3. Temporal clustering: the spray happens within a short time window.
  4. Low per-IP, high global failed-user count.

CLARIFYING QUESTIONS
--------------------
1. Input format?
   → Dicts: {ts, src_ip, username, result: "success"/"failure", password (optional)}
2. Should we track per-IP attempts, or global signal?
   → Both; spray is detectable only by aggregating across IPs.
3. Is the password field available?
   → Often not in real logs; design for absence; use when available.
4. What window size?
   → Rolling 1-hour window (configurable).
5. Thresholds?
   → Global: ≥10 distinct IPs each attempting ≤3 users → spray signal.
     Configurable: min_ips, max_attempts_per_ip, window_seconds.
6. Should successful logins during a spray count as a compromise indicator?
   → Yes; track which users succeeded during an active spray window.
7. Distributed?
   → Each collector reports to a central aggregator (show how to merge).

=============================================================================
APPROACH 1 — NAIVE (two-level dict, no eviction)
=============================================================================
  spray_data[ip][username] = list of timestamps
Query: find IPs with |users| ≤ max_per_ip; count distinct such IPs.
  Time : O(1) per event, O(I×U) per query  (I IPs, U users per IP)
  Space: O(I×U×T) — unbounded

=============================================================================
APPROACH 2 — SLIDING WINDOW + HYPERLOGLOG per IP  (RECOMMENDED)
=============================================================================
For the spray detector, the key question is:
  "How many distinct IPs attempted auth failures in this window where each IP
   hit ≤ max_attempts_per_ip distinct users?"

Design:
  - Per-IP tracker: sliding deque of (ts, username) pairs + exact set of
    usernames in window (small set; ≤ max threshold).
  - Global spray aggregator:
    - HyperLogLog for "distinct users attacked across all IPs in window" (scale)
    - Exact counter for "distinct qualifying IPs" (IPs at ≤ threshold)

When global qualifying IP count ≥ min_ips AND global distinct users ≥ global_user_threshold:
  → Spray alert.

  Time : O(1) per event (amortised with eviction)
  Space: O(events in window) per IP; O(HLL size) for global counters

=============================================================================
APPROACH 3 — BLOOM FILTER for seen (IP, username) pairs  (memory optimised)
=============================================================================
Use a Bloom filter to deduplicate (IP, username) pairs seen within the window.
Then maintain a per-IP counter and a per-username counter.
False positive rate of Bloom filter means we may slightly over-count, but
for spray detection the directional signal remains valid.

=============================================================================
"""

from __future__ import annotations

import hashlib
import math
from collections import defaultdict, deque
from dataclasses import dataclass, field
from datetime import datetime
from typing import Dict, List, Optional, Set, Tuple


# ---------------------------------------------------------------------------
# Data Types
# ---------------------------------------------------------------------------

@dataclass
class SprayAlert:
    window_start: float
    window_end: float
    qualifying_ip_count: int     # IPs with ≤ max_attempts_per_ip users
    distinct_user_count: int     # total distinct users targeted in window
    compromised_users: List[str] # users who succeeded during the spray
    sample_ips: List[str]
    sample_users: List[str]
    confidence: float            # 0–1


# ---------------------------------------------------------------------------
# Helpers
# ---------------------------------------------------------------------------

def _parse_ts(ts) -> Optional[float]:
    if isinstance(ts, (int, float)):
        return float(ts)
    try:
        return datetime.fromisoformat(str(ts).replace("Z", "+00:00")).timestamp()
    except (ValueError, TypeError):
        return None


# ---------------------------------------------------------------------------
# APPROACH 1 — Naive (dict of dicts, no eviction)
# ---------------------------------------------------------------------------

class NaivePasswordSprayDetector:
    """
    Stores all events.  At query time, counts qualifying IPs.
    O(1) per insert; O(I×U) per query.
    """

    def __init__(
        self,
        max_attempts_per_ip: int = 3,
        min_qualifying_ips: int = 10,
        global_user_threshold: int = 20,
    ) -> None:
        self.max_per_ip = max_attempts_per_ip
        self.min_ips = min_qualifying_ips
        self.global_user_threshold = global_user_threshold
        # ip → {username: count}
        self._fail: Dict[str, Dict[str, int]] = defaultdict(lambda: defaultdict(int))
        # {username → True} for success tracking
        self._success: Dict[str, Set[str]] = defaultdict(set)  # user → set of ips
        self.malformed = 0

    def process(self, event: dict) -> None:
        ts = _parse_ts(event.get("ts", 0))
        src = event.get("src_ip") or event.get("src")
        user = event.get("username") or event.get("user")
        result = event.get("result", "failure").lower()

        if not src or not user:
            self.malformed += 1
            return

        if result == "failure":
            self._fail[src][user] += 1
        elif result == "success":
            self._success[user].add(src)

    def check_spray(self) -> Optional[SprayAlert]:
        qualifying_ips = [ip for ip, users in self._fail.items()
                          if len(users) <= self.max_per_ip]
        all_targeted_users: Set[str] = set()
        for ip in qualifying_ips:
            all_targeted_users.update(self._fail[ip].keys())

        if (len(qualifying_ips) >= self.min_ips
                and len(all_targeted_users) >= self.global_user_threshold):
            compromised = [u for u in all_targeted_users if self._success.get(u)]
            return SprayAlert(
                window_start=0, window_end=0,
                qualifying_ip_count=len(qualifying_ips),
                distinct_user_count=len(all_targeted_users),
                compromised_users=compromised,
                sample_ips=qualifying_ips[:10],
                sample_users=sorted(all_targeted_users)[:10],
                confidence=min(1.0, len(qualifying_ips) / (self.min_ips * 2)),
            )
        return None


# ---------------------------------------------------------------------------
# Per-IP Sliding Window Tracker
# ---------------------------------------------------------------------------

class _IPWindow:
    """
    Tracks failed auth attempts per IP within a rolling time window.
    Stores (ts, username) deque for eviction; set for O(1) unique count.
    """

    def __init__(self, window_seconds: float, max_per_ip: int) -> None:
        self.window = window_seconds
        self.max_per_ip = max_per_ip
        self._queue: deque = deque()        # (ts, username)
        self._user_counts: Dict[str, int] = defaultdict(int)

    def add(self, ts: float, username: str) -> None:
        self._evict(ts)
        self._queue.append((ts, username))
        self._user_counts[username] += 1

    def _evict(self, now: float) -> None:
        cutoff = now - self.window
        while self._queue and self._queue[0][0] < cutoff:
            _, old_user = self._queue.popleft()
            self._user_counts[old_user] -= 1
            if self._user_counts[old_user] <= 0:
                del self._user_counts[old_user]

    def is_qualifying(self) -> bool:
        """True if this IP targeted ≤ max_per_ip distinct users (spray-like)."""
        return 1 <= len(self._user_counts) <= self.max_per_ip

    def unique_users(self) -> Set[str]:
        return set(self._user_counts.keys())

    def total_attempts(self) -> int:
        return len(self._queue)


# ---------------------------------------------------------------------------
# APPROACH 2 — Sliding Window + Global Aggregator  (RECOMMENDED)
# ---------------------------------------------------------------------------

class SlidingWindowSprayDetector:
    """
    Streaming distributed password spray detector.

    Per-IP tracker:  SlidingWindowCounter for usernames.
    Global tracker:  Deque of qualifying IP events for spray window counting.

    Alert condition:
      In any window_seconds window:
        - ≥ min_qualifying_ips IPs each with ≤ max_per_ip distinct targeted users
        - Total distinct users across qualifying IPs ≥ global_user_threshold

    Emits a SprayAlert at most once per suppress_seconds period (dedup).
    """

    def __init__(
        self,
        window_seconds: float = 3600.0,
        max_per_ip: int = 3,
        min_qualifying_ips: int = 10,
        global_user_threshold: int = 20,
        suppress_seconds: float = 300.0,
    ) -> None:
        self.window = window_seconds
        self.max_per_ip = max_per_ip
        self.min_ips = min_qualifying_ips
        self.global_user_threshold = global_user_threshold
        self.suppress = suppress_seconds

        # Per-IP windows (failure events only)
        self._ip_windows: Dict[str, _IPWindow] = {}
        # Track success events for compromise detection
        self._success: Dict[str, Set[str]] = defaultdict(set)  # user → ips
        # For global deduplication: when did we last fire an alert?
        self._last_alert_ts: float = 0.0
        self.malformed = 0

    def process(self, event: dict) -> Optional[SprayAlert]:
        ts = _parse_ts(event.get("ts", 0))
        src = event.get("src_ip") or event.get("src")
        user = event.get("username") or event.get("user")
        result = event.get("result", "failure").lower()

        if not src or not user or ts is None:
            self.malformed += 1
            return None

        if result == "success":
            self._success[user].add(src)
            return None

        # Failure path
        if src not in self._ip_windows:
            self._ip_windows[src] = _IPWindow(self.window, self.max_per_ip)
        self._ip_windows[src].add(ts, user)

        # Check spray condition (O(qualifying IPs) at most)
        return self._check(ts)

    def _check(self, now: float) -> Optional[SprayAlert]:
        if now - self._last_alert_ts < self.suppress:
            return None

        # Collect qualifying IPs and aggregate their targeted users
        qualifying_ips = []
        all_users: Set[str] = set()

        for ip, win in self._ip_windows.items():
            win._evict(now)
            if win.is_qualifying():
                qualifying_ips.append(ip)
                all_users.update(win.unique_users())

        if (len(qualifying_ips) >= self.min_ips
                and len(all_users) >= self.global_user_threshold):
            self._last_alert_ts = now
            compromised = [u for u in all_users if self._success.get(u)]
            confidence = min(
                1.0,
                (len(qualifying_ips) / self.min_ips) *
                (len(all_users) / self.global_user_threshold) * 0.5
            )
            return SprayAlert(
                window_start=now - self.window,
                window_end=now,
                qualifying_ip_count=len(qualifying_ips),
                distinct_user_count=len(all_users),
                compromised_users=compromised,
                sample_ips=qualifying_ips[:10],
                sample_users=sorted(all_users)[:10],
                confidence=confidence,
            )
        return None

# APPROACH 3 — Bloom Filter-assisted (memory optimised for very large streams)
# ---------------------------------------------------------------------------

class SimpleBloomFilter:
    """Minimal Bloom filter for (IP, user) deduplication."""

    def __init__(self, capacity: int = 100_000, error_rate: float = 0.01) -> None:
        self.size = math.ceil(-capacity * math.log(error_rate) / (math.log(2) ** 2))
        self.hash_count = math.ceil((self.size / capacity) * math.log(2))
        self._bits = bytearray(math.ceil(self.size / 8))

    def _hashes(self, item: str) -> List[int]:
        result = []
        encoded = item.encode()
        for i in range(self.hash_count):
            seed = (i * 0xdeadbeef).to_bytes(4, "little")
            h = int(hashlib.sha256(seed + encoded).hexdigest(), 16)
            result.append(h % self.size)
        return result

    def add(self, item: str) -> None:
        for bit in self._hashes(item):
            self._bits[bit // 8] |= (1 << (bit % 8))

    def might_contain(self, item: str) -> bool:
        return all((self._bits[bit // 8] >> (bit % 8)) & 1
                   for bit in self._hashes(item))


class BloomSprayDetector:
    """
    Bloom-assisted spray detector.  Uses Bloom filter to deduplicate (IP, user)
    pairs before incrementing per-IP and global counters.

    Trade-off: Bloom false positives mean we may under-count unique (IP, user)
    pairs, resulting in slightly conservative (fewer) alerts.
    Suitable for extremely high event rates where exact sets are too large.
    """

    def __init__(
        self,
        window_seconds: float = 3600,
        max_per_ip: int = 3,
        min_qualifying_ips: int = 10,
        global_user_threshold: int = 20,
    ) -> None:
        self.window = window_seconds
        self.max_per_ip = max_per_ip
        self.min_ips = min_qualifying_ips
        self.global_user_threshold = global_user_threshold
        self._bloom = SimpleBloomFilter()
        # ip → set of usernames (small; capped at max_per_ip+1 for memory bound)
        self._ip_users: Dict[str, Set[str]] = defaultdict(set)
        self._global_users: Set[str] = set()
        self.malformed = 0

    def process(self, event: dict) -> Optional[SprayAlert]:
        src = event.get("src_ip") or event.get("src")
        user = event.get("username") or event.get("user")
        result = event.get("result", "failure").lower()
        if not src or not user or result != "failure":
            if not src or not user:
                self.malformed += 1
            return None

        pair = f"{src}|{user}"
        if not self._bloom.might_contain(pair):
            self._bloom.add(pair)
            self._ip_users[src].add(user)
            self._global_users.add(user)

        return self._check()

    def _check(self) -> Optional[SprayAlert]:
        qualifying = [ip for ip, users in self._ip_users.items()
                      if 1 <= len(users) <= self.max_per_ip]
        if (len(qualifying) >= self.min_ips
                and len(self._global_users) >= self.global_user_threshold):
            return SprayAlert(
                window_start=0, window_end=0,
                qualifying_ip_count=len(qualifying),
                distinct_user_count=len(self._global_users),
                compromised_users=[],
                sample_ips=qualifying[:10],
                sample_users=sorted(self._global_users)[:10],
                confidence=0.8,
            )
        return None

# Test Cases

def _make_fail(src: str, user: str, ts: float) -> dict:
    return {"src_ip": src, "username": user, "result": "failure", "ts": ts}

def _make_success(src: str, user: str, ts: float) -> dict:
    return {"src_ip": src, "username": user, "result": "success", "ts": ts}


def test_naive_spray():
    det = NaivePasswordSprayDetector(max_attempts_per_ip=3, min_qualifying_ips=5,
                                      global_user_threshold=10)
    # 10 IPs each hitting 2 users (≤3), total 15 unique users
    for i in range(10):
        det.process(_make_fail(f"10.0.{i}.1", f"user_{i}", 0))
        det.process(_make_fail(f"10.0.{i}.1", f"user_{i+10}", 0))
    result = det.check_spray()
    assert result is not None, "Expected spray alert"
    assert result.qualifying_ip_count == 10
    print(f"NaiveSpray: qualifying_ips={result.qualifying_ip_count} users={result.distinct_user_count}")


def test_naive_no_spray():
    det = NaivePasswordSprayDetector(max_attempts_per_ip=3, min_qualifying_ips=5,
                                      global_user_threshold=10)
    # Only 3 IPs → below threshold
    for i in range(3):
        det.process(_make_fail(f"10.0.{i}.1", f"user_{i}", 0))
    result = det.check_spray()
    assert result is None, "Should not alert with only 3 IPs"
    print("NaiveSpray no-alert: passed")


def test_sliding_spray():
    det = SlidingWindowSprayDetector(
        window_seconds=3600, max_per_ip=3, min_qualifying_ips=5,
        global_user_threshold=8, suppress_seconds=0
    )
    base_ts = 1_000_000.0
    alert = None
    for i in range(10):
        for u in range(2):
            a = det.process(_make_fail(f"10.0.{i}.1", f"user_{i*2+u}", base_ts + i * 5))
            if a:
                alert = a
    assert alert is not None, "Expected spray alert"
    assert alert.qualifying_ip_count >= 5
    print(f"SlidingWindowSpray: ips={alert.qualifying_ip_count} users={alert.distinct_user_count}")


def test_compromise_detection():
    det = SlidingWindowSprayDetector(
        window_seconds=3600, max_per_ip=3, min_qualifying_ips=5,
        global_user_threshold=8, suppress_seconds=0
    )
    base_ts = 1_000_000.0
    # Spray then one user "user_0" succeeds
    for i in range(10):
        for u in range(2):
            det.process(_make_fail(f"10.0.{i}.1", f"user_{i*2+u}", base_ts + i * 5))
    det.process(_make_success("10.0.5.1", "user_0", base_ts + 60))
    alert = det._check(base_ts + 61)
    if alert:
        print(f"Compromised users: {alert.compromised_users}")
    else:
        print("Compromise detection: no alert triggered (threshold not met in this path)")


def test_malformed():
    det = SlidingWindowSprayDetector()
    det.process({})
    det.process({"src_ip": "1.1.1.1"})  # missing user
    assert det.malformed == 2
    print("Malformed: passed")

"""
THREAT MODEL
  Password spraying is the preferred technique for:
  - Cloud tenant compromise (Office 365, GSuite, Okta)
  - VPN credential theft
  - Corporate email takeover

  Attackers use credential stuffing lists + common passwords ("Spring2026!").
  They distribute across many IPs (residential proxies, compromised hosts,
  AWS lambda, Tor) to stay under per-source lockout thresholds.

  Real-world examples: HAFNIUM, APT33, Scattered Spider

KEY SIGNALS
  1. Failure spike: global auth failure rate > baseline × N
  2. Many distinct IPs, each with low per-IP unique user count
  3. Short time window: spray typically completes in <1 hour
  4. Common password hash in failures (if logs include it)

FALSE POSITIVES
  - Misconfigured apps retrying stale credentials → low distinct user count; filter
  - Legitimate load test → usually from internal range; filter by src subnet

FALSE NEGATIVES
  - Very slow spray (1 attempt per IP per day) → extend window to 24h
  - Spray hidden behind legitimate OAuth providers → use application-layer logs

PRODUCTION HARDENING
  - Integrate with identity provider (Okta, Azure AD) APIs for real-time blocking
  - Use adaptive MFA enforcement triggered by spray alerts
  - Emit alert to Splunk / PagerDuty with full evidence
  - Track compromised_users separately for immediate password reset workflows
"""
