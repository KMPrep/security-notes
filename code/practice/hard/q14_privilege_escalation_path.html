"""
=============================================================================
Q14 [Hard] — Privilege Escalation Path Detection
=============================================================================

PROBLEM
-------
Given a directed privilege graph where nodes are principals (users, groups,
roles, service accounts) and edges represent "can escalate to" relationships
(e.g. user A is member of group G, group G can sudo to root, user B can
impersonate service account S, etc.), detect:

  1. Any path from an unprivileged user to a high-privilege principal
     (shortest path = most likely attack route).
  2. Cycles in the privilege graph (misconfigured mutual escalation).
  3. Diamond paths (multiple independent escalation routes to the same target).
  4. Service principals with excessive reach (high out-degree in privilege graph).

CLARIFYING QUESTIONS
--------------------
1. What types of edges exist?
   → member_of, sudo_to, impersonate, assume_role, token_grant, run_as
2. What defines a "high-privilege" node?
   → Configurable set: {"root", "domain_admin", "cluster-admin", "aws:iam:admin"}
3. Are edges weighted?
   → Optional: edge has a "risk_score" (higher = easier to exploit). Dijkstra for min-risk.
4. Is the graph static (baseline) or streamed?
   → Both: load a baseline, then process delta events.
5. Should we report all paths or just shortest?
   → Shortest first, then optionally enumerate k-shortest.
6. Scale?
   → ~100K principals for a large enterprise — BFS/Dijkstra must be O(V + E log V).
7. Can edges be revoked?
   → Yes; support remove_edge() for JIT access scenarios.

=============================================================================
APPROACH 1 — NAIVE (DFS from every user to every target)
=============================================================================
For each unprivileged user, DFS to all high-privilege nodes.
  Time : O(U × (V + E))  — U = unprivileged users, V/E = graph size
  Space: O(V + E)

=============================================================================
APPROACH 2 — REVERSE BFS from high-privilege targets (RECOMMENDED)
=============================================================================
Run BFS backwards from each high-privilege node.
Any node that can reach a high-privilege node = at-risk principal.
Then extract shortest paths for each at-risk user.

  Time : O(T × (V + E))  — T = target nodes (usually small, e.g. 5-20)
  Space: O(V) for visited + predecessor map

For weighted edges: reverse Dijkstra from targets.
  Time : O((V + E) log V) per target

=============================================================================
APPROACH 3 — INCREMENTAL UPDATE + TOPOLOGICAL SORT for cycle detection
=============================================================================
On each edge addition:
  - DFS for cycles (Kahn's algorithm on the affected component).
  - Update BFS distance from all high-privilege nodes.
  Amortised O(V + E) over all updates.

=============================================================================
"""

from __future__ import annotations

import heapq
from collections import defaultdict, deque
from dataclasses import dataclass, field
from typing import Dict, FrozenSet, List, Optional, Set, Tuple

# ---------------------------------------------------------------------------
# Default high-privilege targets
# ---------------------------------------------------------------------------

DEFAULT_HIGH_PRIV: FrozenSet[str] = frozenset({
    "root", "domain_admin", "enterprise_admin", "cluster-admin",
    "aws:iam:admin", "gcp:owner", "azure:owner", "svc_root",
    "SYSTEM",
})


# ---------------------------------------------------------------------------
# Data Types
# ---------------------------------------------------------------------------

@dataclass
class EscalationEdge:
    src: str
    dst: str
    edge_type: str   # member_of | sudo_to | impersonate | assume_role
    risk_score: float = 1.0  # lower = easier to exploit


@dataclass
class EscalationPath:
    user: str
    target: str
    path: List[str]
    total_risk: float        # sum of edge risk scores along path
    edge_types: List[str]


@dataclass
class GraphAlert:
    alert_type: str   # escalation_path | cycle | excessive_reach
    subject: str
    detail: str
    path: List[str] = field(default_factory=list)
    severity: str = "high"


# ---------------------------------------------------------------------------
# APPROACH 1 — Naive DFS
# ---------------------------------------------------------------------------

class NaivePrivEscDetector:
    """
    DFS from each non-privileged user to find paths to high-privilege nodes.
    O(U × (V + E)) — fine for small graphs / offline analysis.
    """

    def __init__(
        self,
        high_priv_nodes: FrozenSet[str] = DEFAULT_HIGH_PRIV,
    ) -> None:
        self.high_priv = high_priv_nodes
        self._adj: Dict[str, List[EscalationEdge]] = defaultdict(list)
        self._all_nodes: Set[str] = set()

    def add_edge(self, src: str, dst: str, edge_type: str = "member_of",
                 risk_score: float = 1.0) -> None:
        e = EscalationEdge(src, dst, edge_type, risk_score)
        self._adj[src].append(e)
        self._all_nodes.update([src, dst])

    def remove_edge(self, src: str, dst: str, edge_type: str) -> None:
        self._adj[src] = [e for e in self._adj[src]
                          if not (e.dst == dst and e.edge_type == edge_type)]

    def find_all_paths(self, start: str, target: str) -> List[EscalationPath]:
        """DFS to find all simple paths from start to target."""
        results = []

        def dfs(node, path, types, risk, visited):
            if node == target:
                results.append(EscalationPath(
                    user=start, target=target,
                    path=list(path), total_risk=risk,
                    edge_types=list(types),
                ))
                return
            for edge in self._adj.get(node, []):
                if edge.dst not in visited:
                    visited.add(edge.dst)
                    dfs(edge.dst, path + [edge.dst], types + [edge.edge_type],
                        risk + edge.risk_score, visited)
                    visited.discard(edge.dst)

        dfs(start, [start], [], 0.0, {start})
        return results

    def detect_escalations(self) -> List[GraphAlert]:
        alerts = []
        unprivileged = self._all_nodes - self.high_priv
        for user in unprivileged:
            for target in self.high_priv:
                paths = self.find_all_paths(user, target)
                for p in paths:
                    alerts.append(GraphAlert(
                        alert_type="escalation_path",
                        subject=user,
                        detail=f"{user} → {target} via {' → '.join(p.path)}",
                        path=p.path, severity="critical",
                    ))
        return alerts

    def find_cycles(self) -> List[List[str]]:
        """Find all cycles using DFS with colour marking."""
        WHITE, GRAY, BLACK = 0, 1, 2
        colour = defaultdict(int)
        parent = {}
        cycles = []

        def dfs(node):
            colour[node] = GRAY
            for edge in self._adj.get(node, []):
                nb = edge.dst
                if colour[nb] == GRAY:
                    # Reconstruct cycle
                    cycle = [nb]
                    cur = node
                    while cur != nb:
                        cycle.append(cur)
                        cur = parent.get(cur, nb)
                    cycle.append(nb)
                    cycles.append(list(reversed(cycle)))
                elif colour[nb] == WHITE:
                    parent[nb] = node
                    dfs(nb)
            colour[node] = BLACK

        for node in list(self._all_nodes):
            if colour[node] == WHITE:
                dfs(node)
        return cycles


# ---------------------------------------------------------------------------
# APPROACH 2 — Reverse BFS / Dijkstra from high-privilege targets
# ---------------------------------------------------------------------------

class PrivEscDetector:
    """
    Efficient privilege escalation detector.

    Strategy: reverse BFS/Dijkstra from each high-privilege target.
    This computes, for every principal in the graph, the minimum-risk
    path to any high-privilege node.

    Then any unprivileged user with a path is flagged.
    """

    def __init__(
        self,
        high_priv_nodes: FrozenSet[str] = DEFAULT_HIGH_PRIV,
    ) -> None:
        self.high_priv = high_priv_nodes
        # Forward adjacency
        self._adj: Dict[str, List[EscalationEdge]] = defaultdict(list)
        # Reverse adjacency (dst → src edges)
        self._rev: Dict[str, List[EscalationEdge]] = defaultdict(list)
        self._all_nodes: Set[str] = set()

    def add_edge(self, src: str, dst: str, edge_type: str = "member_of",
                 risk_score: float = 1.0) -> None:
        e = EscalationEdge(src, dst, edge_type, risk_score)
        self._adj[src].append(e)
        self._rev[dst].append(e)
        self._all_nodes.update([src, dst])

    def remove_edge(self, src: str, dst: str, edge_type: str) -> None:
        self._adj[src] = [e for e in self._adj[src]
                          if not (e.dst == dst and e.edge_type == edge_type)]
        self._rev[dst] = [e for e in self._rev[dst]
                          if not (e.src == src and e.edge_type == edge_type)]

    def shortest_paths_to_target(self, target: str) -> Dict[str, EscalationPath]:
        """
        Reverse Dijkstra from target; returns {user: shortest EscalationPath}.
        dist[node] = min risk score to reach target from node.
        """
        # dist[node] = (total_risk, predecessor, edge_type)
        dist: Dict[str, float] = {target: 0.0}
        pred: Dict[str, Tuple[str, str]] = {}  # node → (prev_node, edge_type)
        heap: List[Tuple[float, str]] = [(0.0, target)]

        while heap:
            d, node = heapq.heappop(heap)
            if d > dist.get(node, float("inf")):
                continue
            # Walk reverse edges: who has an edge pointing TO this node?
            for edge in self._rev.get(node, []):
                new_dist = d + edge.risk_score
                if new_dist < dist.get(edge.src, float("inf")):
                    dist[edge.src] = new_dist
                    pred[edge.src] = (node, edge.edge_type)
                    heapq.heappush(heap, (new_dist, edge.src))

        # Reconstruct paths
        paths = {}
        for node, d in dist.items():
            if node == target:
                continue
            path = [node]
            types = []
            cur = node
            while cur != target:
                nxt, etype = pred.get(cur, (None, None))
                if nxt is None:
                    break
                path.append(nxt)
                types.append(etype)
                cur = nxt
            paths[node] = EscalationPath(
                user=node, target=target,
                path=path, total_risk=d, edge_types=types,
            )
        return paths

    def detect_escalations(self) -> List[GraphAlert]:
        """Find all unprivileged users with a path to any high-priv node."""
        alerts = []
        seen: Set[Tuple[str, str]] = set()

        for target in self.high_priv:
            if target not in self._all_nodes:
                continue
            paths = self.shortest_paths_to_target(target)
            for user, ep in paths.items():
                if user in self.high_priv:
                    continue
                key = (user, target)
                if key not in seen:
                    seen.add(key)
                    alerts.append(GraphAlert(
                        alert_type="escalation_path",
                        subject=user,
                        detail=(f"Risk {ep.total_risk:.1f}: "
                                f"{' → '.join(ep.path)}"),
                        path=ep.path, severity="critical",
                    ))
        return alerts

    def detect_cycles(self) -> List[GraphAlert]:
        """Kahn's algorithm (BFS topological sort) — cycle = remaining nodes."""
        in_degree: Dict[str, int] = defaultdict(int)
        for src, edges in self._adj.items():
            for e in edges:
                in_degree[e.dst] += 1
        queue = deque(n for n in self._all_nodes if in_degree[n] == 0)
        processed: Set[str] = set()
        while queue:
            node = queue.popleft()
            processed.add(node)
            for e in self._adj.get(node, []):
                in_degree[e.dst] -= 1
                if in_degree[e.dst] == 0:
                    queue.append(e.dst)
        cyclic_nodes = self._all_nodes - processed
        alerts = []
        if cyclic_nodes:
            alerts.append(GraphAlert(
                alert_type="cycle",
                subject=str(cyclic_nodes),
                detail=f"Privilege graph cycle involving: {cyclic_nodes}",
                severity="high",
            ))
        return alerts

    def high_reach_principals(self, threshold: int = 10) -> List[GraphAlert]:
        """Find nodes with out-degree (forward or transitive) > threshold."""
        alerts = []
        for node, edges in self._adj.items():
            if len(edges) > threshold:
                alerts.append(GraphAlert(
                    alert_type="excessive_reach",
                    subject=node,
                    detail=f"{node} has {len(edges)} direct privilege edges",
                ))
        return alerts

# Test Cases
# ---------------------------------------------------------------------------

def test_naive_simple_path():
    det = NaivePrivEscDetector()
    det.add_edge("alice", "dev-group", "member_of")
    det.add_edge("dev-group", "sudo_users", "member_of")
    det.add_edge("sudo_users", "root", "sudo_to")
    alerts = det.detect_escalations()
    assert any("alice" in a.subject and "root" in a.path for a in alerts), \
        f"Expected alice→root path; got {alerts}"
    print("Naive simple path: passed")


def test_naive_cycle():
    det = NaivePrivEscDetector()
    det.add_edge("A", "B", "member_of")
    det.add_edge("B", "C", "member_of")
    det.add_edge("C", "A", "member_of")
    cycles = det.find_cycles()
    assert len(cycles) > 0, "Expected a cycle"
    print("Naive cycle detection: passed")


def test_reverse_dijkstra_path():
    det = PrivEscDetector()
    det.add_edge("alice", "dev-group", "member_of", risk_score=1.0)
    det.add_edge("dev-group", "root", "sudo_to", risk_score=2.0)
    det.add_edge("alice", "root", "sudo_to", risk_score=0.5)  # direct, lower risk
    alerts = det.detect_escalations()
    alice_alerts = [a for a in alerts if a.subject == "alice"]
    assert len(alice_alerts) == 1  # one alert per (user, target)
    # Shortest path: alice → root directly (risk 0.5)
    ep = det.shortest_paths_to_target("root")["alice"]
    assert ep.total_risk == pytest_approx_soft(0.5, ep.total_risk)
    print("Reverse Dijkstra path: passed")


def pytest_approx_soft(expected, actual, tol=1e-6):
    return abs(actual - expected) < tol


def test_no_path():
    det = PrivEscDetector()
    det.add_edge("alice", "dev-group", "member_of")
    # dev-group does not reach any high-priv node → no alerts
    alerts = det.detect_escalations()
    assert len(alerts) == 0
    print("No path: passed")


def test_cycle_detection():
    det = PrivEscDetector()
    det.add_edge("A", "B", "member_of")
    det.add_edge("B", "C", "member_of")
    det.add_edge("C", "A", "member_of")  # cycle
    alerts = det.detect_cycles()
    assert len(alerts) > 0
    assert alerts[0].alert_type == "cycle"
    print("Kahn cycle detection: passed")


def test_remove_edge():
    det = PrivEscDetector()
    det.add_edge("bob", "root", "sudo_to")
    det.remove_edge("bob", "root", "sudo_to")
    alerts = det.detect_escalations()
    assert all(a.subject != "bob" for a in alerts)
    print("Remove edge: passed")


def test_excessive_reach():
    det = PrivEscDetector()
    for i in range(15):
        det.add_edge("svc_super", f"target-{i}", "assume_role")
    alerts = det.high_reach_principals(threshold=10)
    assert any(a.subject == "svc_super" for a in alerts)
    print("Excessive reach: passed")

"""
THREAT MODEL
  Privilege escalation paths are the map an attacker follows after initial
  compromise.  Common real-world paths:

  1. developer → dev-group → prod-deploy → AWS IAM admin
  2. webapp-service-account → K8s RBAC cluster-admin  (wildcard permissions)
  3. user → sudo NOPASSWD → /bin/bash → root
  4. lower-priv role → iam:PassRole → high-priv role → full AWS access

  Graph analysis reveals these paths before an attacker does.

PRODUCTION USE CASES
  - Run this detector after every IAM/RBAC change (CI/CD gate).
  - Visualise the graph with Cytoscape or Neo4j for analyst review.
  - Integrate with Terraform plan output: flag new paths created by a PR.
  - Use k-shortest-paths (Yen's algorithm) to find alternative routes an
    attacker could take if the primary path is blocked.

FAILURE MODES
  - Incomplete graph: edges from external IdP not modelled → missed paths.
  - Transitive wildcards: AWS IAM policy with "*" → all resources / all actions;
    requires policy simulator, not just graph edges.
  - Temporal access: JIT access grants temporary edges; model as ephemeral edges
    with expiry, run detection continuously during the JIT window.
  - Service account impersonation via token projection (Kubernetes): not a
    traditional "member_of" edge; requires specialised modelling.

TOOLS IN THE WILD
  - BloodHound (Active Directory privilege graph)
  - Principal Mapper (AWS IAM graph)
  - Cartography (multi-cloud asset graph with escalation edges)
"""
