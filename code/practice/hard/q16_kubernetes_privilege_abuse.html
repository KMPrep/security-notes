"""
=============================================================================
Q16 [Hard] — Kubernetes Privilege Abuse Detection
=============================================================================

PROBLEM
-------
Kubernetes audit logs record every API call to the API server.
Detect the following privilege abuse patterns from audit events:

  1. RBAC Escalation:  creating or modifying (Cluster)RoleBindings or
     (Cluster)Roles that grant highly-sensitive permissions.
  2. Exec/Attach to Pod: kubectl exec / attach into pods — unusual for prod.
  3. Privileged Pod Creation: pod with securityContext.privileged=true, or
     hostPID/hostNetwork/hostIPC=true, or dangerous capabilities.
  4. Secret Enumeration: listing or getting Secrets across many namespaces.
  5. Wildcard permissions: rules with "*" in verbs or resources.
  6. Service Account Token Mounting: pod automountServiceAccountToken=true
     with a high-privilege SA (detected post-hoc from RBAC bindings).
  7. Namespace creation surge: many new namespaces in a short window
     (attacker trying to evade per-namespace NetworkPolicy / LimitRange).

CLARIFYING QUESTIONS
--------------------
1. Input: raw Kubernetes audit log lines (JSON)?
   → Yes; subset of the full K8s audit schema.
2. How to detect "high-privilege" role/binding?
   → Check for wildcard verbs/resources or specific dangerous combos:
     (verbs=[*], resources=[*]), (verbs=[get,list,watch], resources=[secrets]),
     (verbs=[create,update], resources=[clusterrolebindings]).
3. Should exec alerts distinguish between interactive shell and one-off commands?
   → Flag all exec/attach; note if stdin=true (interactive shell).
4. How to detect privileged pod without the full pod spec?
   → Look at requestObject.spec.containers[*].securityContext and pod-level
     spec.hostPID / spec.hostNetwork / spec.hostIPC fields.
5. Allowlist for known-good principals?
   → Yes; accept set of trusted usernames/SA names.
6. Multi-namespace secret enumeration threshold?
   → Configurable; default > 3 distinct namespaces in 60 seconds.

=============================================================================
APPROACH 1 — NAIVE (if-elif chain per event)
=============================================================================
Simple if/elif checks on verb + resource.
  No burst detection, no stateful tracking.
  O(1) per event.

=============================================================================
APPROACH 2 — DISPATCH TABLE + STATEFUL COUNTERS (RECOMMENDED)
=============================================================================
  - Dict of (verb, resource) → handler function for O(1) dispatch
  - Sliding window counters per principal for burst/enumeration detection
  - RBAC analyzer: scan role rules for dangerous permission combos
  O(1) per event (plus O(rules) for RBAC analysis).

=============================================================================
"""

from __future__ import annotations

from collections import defaultdict, deque
from dataclasses import dataclass, field
from datetime import datetime
from typing import Any, Dict, FrozenSet, List, Optional, Set, Tuple

# ---------------------------------------------------------------------------
# Dangerous permissions definitions
# ---------------------------------------------------------------------------

# (verb, resource) tuples that are considered high-privilege
DANGEROUS_PERMS: FrozenSet[Tuple[str, str]] = frozenset({
    ("*", "*"),
    ("*", "secrets"),
    ("create", "clusterrolebindings"),
    ("update", "clusterrolebindings"),
    ("patch", "clusterrolebindings"),
    ("create", "rolebindings"),
    ("update", "rolebindings"),
    ("patch", "roles"),
    ("patch", "clusterroles"),
    ("create", "clusterroles"),
    ("update", "clusterroles"),
    ("escalate", "clusterroles"),
    ("bind", "clusterroles"),
    ("impersonate", "serviceaccounts"),
    ("create", "pods/exec"),
    ("list", "secrets"),
    ("get", "secrets"),
})

# ---------------------------------------------------------------------------
# Data Types
# ---------------------------------------------------------------------------

@dataclass
class K8sAuditEvent:
    event_id: str
    ts: float
    verb: str             # get | list | create | update | patch | delete | exec | attach
    resource: str         # pods | secrets | clusterrolebindings | …
    namespace: str
    name: str             # resource name (if applicable)
    username: str
    user_groups: List[str]
    source_ip: str
    stage: str            # RequestReceived | ResponseStarted | ResponseComplete | Panic
    response_code: int
    request_object: dict = field(default_factory=dict)
    subresource: str = ""

@dataclass
class K8sAlert:
    alert_type: str
    severity: str
    username: str
    verb: str
    resource: str
    namespace: str
    ts: float
    detail: str
    event_id: str


def _parse_ts(ts) -> Optional[float]:
    if isinstance(ts, (int, float)):
        return float(ts)
    try:
        return datetime.fromisoformat(str(ts).replace("Z", "+00:00")).timestamp()
    except (ValueError, TypeError):
        return None


def _parse_event(raw: dict) -> Optional[K8sAuditEvent]:
    ts = _parse_ts(raw.get("requestReceivedTimestamp") or raw.get("ts"))
    if ts is None:
        return None
    obj_ref = raw.get("objectRef", {})
    user = raw.get("user", {})
    return K8sAuditEvent(
        event_id=raw.get("auditID", "unknown"),
        ts=ts,
        verb=raw.get("verb", ""),
        resource=obj_ref.get("resource", ""),
        namespace=obj_ref.get("namespace", ""),
        name=obj_ref.get("name", ""),
        username=user.get("username", "unknown"),
        user_groups=user.get("groups", []),
        source_ip=raw.get("sourceIPs", [""])[0] if raw.get("sourceIPs") else "",
        stage=raw.get("stage", ""),
        response_code=raw.get("responseStatus", {}).get("code", 0),
        request_object=raw.get("requestObject") or {},
        subresource=obj_ref.get("subresource", ""),
    )


# ---------------------------------------------------------------------------
# RBAC Rule Analyser
# ---------------------------------------------------------------------------

def _analyse_rbac_rules(rules: List[dict]) -> List[str]:
    """
    Given a list of RBAC rule dicts (from Role/ClusterRole spec.rules),
    return list of dangerous permission descriptions found.
    """
    findings = []
    for rule in rules:
        verbs = set(rule.get("verbs", []))
        resources = set(rule.get("resources", []))
        for v in verbs:
            for r in resources:
                if (v, r) in DANGEROUS_PERMS or v == "*" or r == "*":
                    findings.append(f"verb={v!r} resource={r!r}")
    return findings


# ---------------------------------------------------------------------------
# APPROACH 1 — Naive detector
# ---------------------------------------------------------------------------

class NaiveK8sAbuseDetector:
    """Simple stateless detector.  O(1) per event."""

    def __init__(
        self,
        allowlist: Optional[Set[str]] = None,
    ) -> None:
        self.allowlist = allowlist or set()
        self.malformed = 0

    def process(self, raw: dict) -> List[K8sAlert]:
        evt = _parse_event(raw)
        if evt is None:
            self.malformed += 1
            return []
        if evt.username in self.allowlist:
            return []
        # Only process completed events with success responses
        if evt.stage not in ("ResponseComplete", "") or evt.response_code >= 400:
            return []

        alerts = []

        # exec/attach
        if evt.subresource in ("exec", "attach"):
            alerts.append(K8sAlert(
                alert_type="pod_exec", severity="high",
                username=evt.username, verb=evt.verb,
                resource="pods/" + evt.subresource, namespace=evt.namespace,
                ts=evt.ts,
                detail=f"{evt.username} exec/attach into pod {evt.name!r}",
                event_id=evt.event_id,
            ))

        # Privileged pod
        if evt.verb in ("create", "update") and evt.resource == "pods":
            findings = _check_privileged_pod(evt.request_object)
            if findings:
                alerts.append(K8sAlert(
                    alert_type="privileged_pod", severity="critical",
                    username=evt.username, verb=evt.verb,
                    resource="pods", namespace=evt.namespace,
                    ts=evt.ts,
                    detail=f"Privileged pod: {findings}",
                    event_id=evt.event_id,
                ))

        # RBAC escalation
        if evt.resource in ("roles", "clusterroles", "rolebindings", "clusterrolebindings"):
            if evt.verb in ("create", "update", "patch"):
                rules = (evt.request_object.get("rules") or
                         evt.request_object.get("roleRef", {}) and [])
                if evt.resource in ("rolebindings", "clusterrolebindings"):
                    role_ref = evt.request_object.get("roleRef", {})
                    alerts.append(K8sAlert(
                        alert_type="rbac_binding", severity="high",
                        username=evt.username, verb=evt.verb,
                        resource=evt.resource, namespace=evt.namespace,
                        ts=evt.ts,
                        detail=f"RBAC binding to {role_ref.get('name', '?')}",
                        event_id=evt.event_id,
                    ))
                elif isinstance(rules, list):
                    findings = _analyse_rbac_rules(rules)
                    if findings:
                        alerts.append(K8sAlert(
                            alert_type="rbac_escalation", severity="critical",
                            username=evt.username, verb=evt.verb,
                            resource=evt.resource, namespace=evt.namespace,
                            ts=evt.ts,
                            detail=f"Dangerous RBAC rule: {findings}",
                            event_id=evt.event_id,
                        ))

        return alerts


def _check_privileged_pod(spec: dict) -> List[str]:
    """Check pod spec for dangerous privilege settings."""
    findings = []
    if spec.get("hostPID"):
        findings.append("hostPID=true")
    if spec.get("hostNetwork"):
        findings.append("hostNetwork=true")
    if spec.get("hostIPC"):
        findings.append("hostIPC=true")
    for container in spec.get("containers", []):
        sc = container.get("securityContext", {})
        if sc.get("privileged"):
            findings.append(f"container {container.get('name','?')!r} privileged=true")
        caps = sc.get("capabilities", {})
        dangerous_caps = {"SYS_ADMIN", "SYS_PTRACE", "NET_ADMIN", "DAC_OVERRIDE"}
        added = set(caps.get("add", [])) & dangerous_caps
        if added:
            findings.append(f"dangerous caps: {added}")
    return findings


# ---------------------------------------------------------------------------
# APPROACH 2 — Full stateful detector (RECOMMENDED)
# ---------------------------------------------------------------------------

class K8sAbuseDetector:
    """
    Full stateful K8s privilege abuse detector with burst tracking.
    """

    def __init__(
        self,
        allowlist: Optional[Set[str]] = None,
        secret_enum_threshold: int = 3,
        secret_enum_window_s: float = 60.0,
        ns_burst_threshold: int = 5,
        ns_burst_window_s: float = 120.0,
        exec_alert_for_system_accounts: bool = False,
    ) -> None:
        self.allowlist = allowlist or set()
        self.secret_enum_threshold = secret_enum_threshold
        self.secret_enum_window = secret_enum_window_s
        self.ns_burst_threshold = ns_burst_threshold
        self.ns_burst_window = ns_burst_window_s
        self.exec_sa = exec_alert_for_system_accounts
        # principal → deque of (ts, namespace) for secret enumeration
        self._secret_enum: Dict[str, deque] = defaultdict(deque)
        # principal → deque of ts for namespace creation burst
        self._ns_create: Dict[str, deque] = defaultdict(deque)
        # Dedup sets
        self._burst_alerted_secret: Set[str] = set()
        self._burst_alerted_ns: Set[str] = set()
        self._naive = NaiveK8sAbuseDetector(allowlist=allowlist)
        self.malformed = 0

    def process(self, raw: dict) -> List[K8sAlert]:
        evt = _parse_event(raw)
        if evt is None:
            self.malformed += 1
            return []
        if evt.username in self.allowlist:
            return []
        if evt.stage not in ("ResponseComplete", "") or evt.response_code >= 400:
            return []

        alerts: List[K8sAlert] = []

        # Run all naive checks first
        alerts.extend(self._naive.process(raw))

        # --- Secret enumeration across namespaces ---
        if evt.resource == "secrets" and evt.verb in ("get", "list", "watch"):
            q = self._secret_enum[evt.username]
            cutoff = evt.ts - self.secret_enum_window
            while q and q[0][0] < cutoff:
                q.popleft()
            q.append((evt.ts, evt.namespace))
            distinct_ns = len({x[1] for x in q})
            if distinct_ns > self.secret_enum_threshold:
                if evt.username not in self._burst_alerted_secret:
                    self._burst_alerted_secret.add(evt.username)
                    alerts.append(K8sAlert(
                        alert_type="secret_enumeration", severity="high",
                        username=evt.username, verb=evt.verb,
                        resource="secrets", namespace="multiple",
                        ts=evt.ts,
                        detail=(f"{evt.username} accessed secrets in {distinct_ns} "
                                f"namespaces within {self.secret_enum_window}s"),
                        event_id=evt.event_id,
                    ))
            else:
                self._burst_alerted_secret.discard(evt.username)

        # --- Namespace creation burst ---
        if evt.resource == "namespaces" and evt.verb == "create":
            q = self._ns_create[evt.username]
            cutoff = evt.ts - self.ns_burst_window
            while q and q[0] < cutoff:
                q.popleft()
            q.append(evt.ts)
            if len(q) > self.ns_burst_threshold:
                if evt.username not in self._burst_alerted_ns:
                    self._burst_alerted_ns.add(evt.username)
                    alerts.append(K8sAlert(
                        alert_type="namespace_burst", severity="medium",
                        username=evt.username, verb="create",
                        resource="namespaces", namespace="",
                        ts=evt.ts,
                        detail=(f"{evt.username} created {len(q)} namespaces "
                                f"in {self.ns_burst_window}s"),
                        event_id=evt.event_id,
                    ))

        return alerts

def _raw(verb, resource, username="alice", ns="default", ts=1000.0,
         subresource="", code=200, req_obj=None) -> dict:
    return {
        "auditID": f"{verb}-{resource}-{ts}",
        "requestReceivedTimestamp": ts,
        "verb": verb,
        "objectRef": {"resource": resource, "namespace": ns,
                      "name": "test-obj", "subresource": subresource},
        "user": {"username": username, "groups": []},
        "sourceIPs": ["10.0.0.1"],
        "stage": "ResponseComplete",
        "responseStatus": {"code": code},
        "requestObject": req_obj or {},
    }

# Test Cases

def test_exec_alert():
    det = K8sAbuseDetector()
    alerts = det.process(_raw("create", "pods", subresource="exec"))
    assert any(a.alert_type == "pod_exec" for a in alerts)
    print("Pod exec: passed")


def test_privileged_pod():
    req = {
        "hostPID": True,
        "containers": [{"name": "c1", "securityContext": {"privileged": True}}],
    }
    det = K8sAbuseDetector()
    alerts = det.process(_raw("create", "pods", req_obj=req))
    assert any(a.alert_type == "privileged_pod" for a in alerts)
    print("Privileged pod: passed")


def test_rbac_binding():
    req = {"roleRef": {"name": "cluster-admin", "kind": "ClusterRole"}}
    det = K8sAbuseDetector()
    alerts = det.process(_raw("create", "clusterrolebindings", req_obj=req))
    assert any(a.alert_type == "rbac_binding" for a in alerts)
    print("RBAC binding: passed")


def test_rbac_wildcard_role():
    req = {"rules": [{"verbs": ["*"], "resources": ["*"]}]}
    det = K8sAbuseDetector()
    alerts = det.process(_raw("create", "clusterroles", req_obj=req))
    assert any(a.alert_type == "rbac_escalation" for a in alerts)
    print("RBAC wildcard role: passed")


def test_secret_enumeration():
    det = K8sAbuseDetector(secret_enum_threshold=2, secret_enum_window_s=60)
    for ns in ["ns1", "ns2", "ns3"]:
        det.process(_raw("list", "secrets", ns=ns, ts=1000.0))
    # Should trigger after 3rd distinct namespace
    final = det.process(_raw("list", "secrets", ns="ns4", ts=1010.0))
    assert any(a.alert_type == "secret_enumeration" for a in final), \
        "Expected secret_enumeration alert"
    print("Secret enumeration: passed")


def test_namespace_burst():
    det = K8sAbuseDetector(ns_burst_threshold=3, ns_burst_window_s=120)
    all_alerts = []
    for i in range(5):
        all_alerts += det.process(_raw("create", "namespaces", ts=float(1000 + i * 5)))
    assert any(a.alert_type == "namespace_burst" for a in all_alerts)
    print("Namespace burst: passed")


def test_failed_events_ignored():
    det = K8sAbuseDetector()
    alerts = det.process(_raw("create", "pods", code=403, subresource="exec"))
    assert len(alerts) == 0
    print("Failed events ignored: passed")


def test_allowlist():
    det = K8sAbuseDetector(allowlist={"alice"})
    alerts = det.process(_raw("create", "pods", subresource="exec"))  # alice allowlisted
    assert len(alerts) == 0
    print("Allowlist: passed")

"""
THREAT MODEL
  Kubernetes RBAC is the authorisation layer for the entire cluster.
  Common attack patterns:

  1. Pod exec → interactive shell inside a running container
     → escape to host via mount, nsenter, or privileged context
  2. Create ClusterRoleBinding to cluster-admin for attacker-controlled SA
     → full cluster takeover with a single API call
  3. List secrets across all namespaces → harvest DB creds, API keys, TLS certs
  4. Create privileged pod mounting hostPath=/ → read/write to worker node filesystem
  5. ServiceAccount token projection → steal JWT and call API server directly

DETECTION STRATEGY
  - Audit all RBAC changes (create/update/patch on roles, bindings).
  - Alert on exec/attach — no legitimate workload needs these in prod.
  - Check pod specs at admission time (OPA/Gatekeeper / Kyverno policies).
  - Flag wildcard verbs/resources in new role definitions.
  - Correlate: new SA + new ClusterRoleBinding + new Pod within 5 minutes
    = automated privilege escalation chain.

HARDENING
  - Disable anonymous auth on API server.
  - Enable audit logging with a policy that captures RequestReceived events.
  - Use Pod Security Admission (PSA) to reject privileged pods.
  - Restrict RBAC: no cluster-admin for developers; use aggregated ClusterRoles.
  - Network policy: limit pods' egress to API server to only necessary SAs.
"""
