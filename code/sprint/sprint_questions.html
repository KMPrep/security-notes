2-Day Practice Sprint: 25 Must-Master Questions

Focus: Core algorithm patterns covering ~80% of first/second round interviews.
Time target: ≤ 20 minutes per problem. Solve naive first, then optimize.
Complexity goal: State time and space complexity for every approach.

Day 1 — Arrays, Strings, Hash Maps

Pattern 1: Arrays & Sliding Window

Q1. Two Sum
Given a list of integers and a target, return the indices of the two numbers that add up to the target.

  • Input: nums = [2, 7, 11, 15], target = 9
  • Output: [0, 1]
  • Constraint: Exactly one solution; same element cannot be used twice.
  • Hint: One-pass HashMap — store value -> index, look up target - num.
  • Complexity goal: O(n) time, O(n) space.

Q2. Longest Substring Without Repeating Characters
Find the length of the longest substring that contains no repeating characters.

  • Input: "abcabcbb"
  • Output: 3 (substring "abc")
  • Hint: Sliding window + dict tracking last-seen index; jump left past the duplicate.
  • Complexity goal: O(n) time, O(min(n, charset)) space.

Q3. Maximum Subarray Sum (Kadane's Algorithm)
Find the contiguous subarray with the largest sum.

  • Input: [-2, 1, -3, 4, -1, 2, 1, -5, 4]
  • Output: 6 (subarray [4, -1, 2, 1])
  • Hint: At each element decide: extend current subarray or start fresh here.
  • Complexity goal: O(n) time, O(1) space.

Q4. Container With Most Water
Given an array of heights, find two lines forming a container holding the most water.

  • Input: [1, 8, 6, 2, 5, 4, 8, 3, 7]
  • Output: 49
  • Hint: Two pointers from both ends; always move the shorter bar inward.
  • Complexity goal: O(n) time, O(1) space.

Q5. Find All Anagrams in a String
Given strings s and p, return the start indices of all anagrams of p in s.

  • Input: s = "cbaebabacd", p = "abc"
  • Output: [0, 6]
  • Hint: Fixed-size sliding window — add incoming char, remove outgoing char, compare freq maps.
  • Complexity goal: O(n) time, O(1) space (26-letter alphabet).

Pattern 2: Strings & Hash Maps

Q6. Valid Parentheses
Given a string of brackets ()[]{}, return True if every open bracket is closed correctly.

  • Input: "({[]})" → Output: True
  • Input: "([)]" → Output: False
  • Hint: Stack — push open brackets, pop and verify on close brackets.
  • Complexity goal: O(n) time, O(n) space.

Q7. Group Anagrams
Given a list of strings, group them by their anagram family.

  • Input: ["eat", "tea", "tan", "ate", "nat", "bat"]
  • Output: [["eat","tea","ate"], ["tan","nat"], ["bat"]]
  • Hint: Sorted string or frequency-tuple as the dict key.
  • Complexity goal: O(n · k log k) time where k = max word length.

Q8. Top K Frequent Elements
Return the k most frequent elements in the array.

  • Input: nums = [1,1,1,2,2,3], k = 2
  • Output: [1, 2]
  • Hint: Counter + min-heap of size k, or bucket sort by frequency for O(n).
  • Complexity goal: O(n log k) heap, O(n) bucket sort.

Q9. Encode and Decode Strings
Design encode/decode so a list of strings survives a round-trip through a single string.

  • Input: ["hello", "world", "foo"]
  • Encoded: e.g. "5#hello5#world3#foo"
  • Decoded: ["hello", "world", "foo"]
  • Constraint: Strings may contain any character including # and spaces.
  • Hint: Length-prefix encoding: <length>#<string> per chunk.

Q10. Minimum Window Substring
Find the shortest substring of s that contains all characters of t.

  • Input: s = "ADOBECODEBANC", t = "ABC"
  • Output: "BANC"
  • Hint: Expand right until all of t is covered, then shrink left as far as possible.
  • Complexity goal: O(n) time, O(|t|) space.

Day 2 — Binary Search, Trees, Dynamic Programming

Pattern 3: Binary Search & Sorting

Q11. Search in Rotated Sorted Array
A sorted array was rotated at an unknown pivot. Return the index of target, or -1.

  • Input: nums = [4, 5, 6, 7, 0, 1, 2], target = 0
  • Output: 4
  • Hint: In any [lo, hi] window, one half is always sorted — determine which, then narrow.
  • Complexity goal: O(log n) time.

Q12. Find First and Last Position of Element
Given a sorted array and target, return [first_index, last_index] or [-1, -1].

  • Input: nums = [5, 7, 7, 8, 8, 10], target = 8
  • Output: [3, 4]
  • Hint: Two binary searches — one biased left, one biased right.
  • Complexity goal: O(log n) time.

Q13. Kth Largest Element in an Array
Find the kth largest element (not kth distinct) in an unsorted array.

  • Input: nums = [3, 2, 1, 5, 6, 4], k = 2
  • Output: 5
  • Hint: Min-heap of size k (O(n log k)), or quickselect for O(n) average.
  • Complexity goal: O(n log k) heap, O(n) average quickselect.

Q14. Merge Intervals
Given a list of intervals, merge all overlapping ones.

  • Input: [[1,3],[2,6],[8,10],[15,18]]
  • Output: [[1,6],[8,10],[15,18]]
  • Hint: Sort by start time, then greedily extend the last merged interval.
  • Complexity goal: O(n log n) time.

Q15. Search a 2D Matrix
Each row is sorted; first int of each row > last int of previous row. Search for target.

  • Input: matrix = [[1,3,5,7],[10,11,16,20],[23,30,34,60]], target = 3
  • Output: True
  • Hint: Treat as 1D sorted array and binary search via mid // n, mid % n.
  • Complexity goal: O(log(m·n)) binary search, O(m+n) staircase.

Pattern 4: Trees & Graphs

Q16. Binary Tree Level-Order Traversal
Return node values grouped by level (BFS).

  • Input: Tree [3, 9, 20, null, null, 15, 7]
  • Output: [[3], [9, 20], [15, 7]]
  • Hint: BFS with a deque; snapshot len(queue) at the start of each level.
  • Complexity goal: O(n) time, O(n) space.

Q17. Validate Binary Search Tree
Determine if a binary tree is a valid BST.

  • Input: Tree [5, 1, 4, null, null, 3, 6]
  • Output: False (4 is in right subtree but 4 < 5)
  • Hint: Pass min/max bounds recursively — every node must be strictly within its range.
  • Complexity goal: O(n) time, O(h) space.

Q18. Lowest Common Ancestor of a BST
Given a BST and two values p and q, find their lowest common ancestor.

  • Input: BST [6,2,8,0,4,7,9], p = 2, q = 8
  • Output: 6
  • Hint: Both < root → go left. Both > root → go right. Otherwise, root is the LCA.
  • Complexity goal: O(h) time where h = tree height.

Q19. Number of Islands
Count the number of islands (connected groups of '1') in a 2D grid.

  • Input: [["1","1","0"],["1","1","0"],["0","0","1"]]
  • Output: 2
  • Hint: DFS or BFS from each unvisited '1'; mark visited cells to avoid recounting.
  • Complexity goal: O(m · n) time and space.

Q20. Clone Graph
Given a node in a connected undirected graph, return a deep copy of the entire graph.

  • Each node has val and neighbors list.
  • Hint: BFS or DFS with a HashMap from original node → cloned node to prevent revisiting.
  • Complexity goal: O(V + E) time and space.

Pattern 5: Dynamic Programming

Q21. Climbing Stairs
You can climb 1 or 2 steps at a time. How many distinct ways to reach step n?

  • Input: n = 5
  • Output: 8
  • Hint: ways(n) = ways(n-1) + ways(n-2) — identical to Fibonacci. Use two variables.
  • Complexity goal: O(n) time, O(1) space.

Q22. Coin Change (Minimum Coins)
Given coin denominations and a target amount, find the fewest coins to make that amount.

  • Input: coins = [1, 5, 11], amount = 15
  • Output: 3 (5 + 5 + 5)
  • Hint: Bottom-up DP: dp[i] = min(dp[i], dp[i - coin] + 1) for each coin ≤ i.
  • Complexity goal: O(amount · len(coins)) time, O(amount) space.

Q23. Longest Increasing Subsequence
Find the length of the longest strictly increasing subsequence.

  • Input: [10, 9, 2, 5, 3, 7, 101, 18]
  • Output: 4 (e.g. [2, 3, 7, 101])
  • Hint: O(n²) DP or O(n log n) with binary search on a sorted tails array.
  • Complexity goal: O(n²) DP, O(n log n) with binary search.

Q24. Word Break
Given string s and a dictionary, determine if s can be segmented into dictionary words.

  • Input: s = "leetcode", wordDict = ["leet", "code"]
  • Output: True
  • Hint: dp[i] = True if s[:i] can be segmented. Check all j < i where dp[j] is True and s[j:i] is in the dict.
  • Complexity goal: O(n²) time, O(n) space.

Q25. House Robber
Cannot rob two adjacent houses. Find the maximum amount you can rob.

  • Input: [2, 7, 9, 3, 1]
  • Output: 12 (2 + 9 + 1)
  • Hint: dp[i] = max(dp[i-1], dp[i-2] + nums[i]). Reduce to two variables.
  • Complexity goal: O(n) time, O(1) space.
